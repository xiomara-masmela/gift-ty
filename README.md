# Gift To You - gift-ty

Tool to help you find a present when you do not know what to buy. It takes 3 parameters what the person likes, if they have a favourite colour and the event. Based on those parameters, the application shows a list of products.


## Technology used

- React
- Node/Express
- MongoDB/Mongoose
- Mui

## Interesting

- Learning Mongo 
- Using the MERN stack


## Roadblocks

- Learning MongoDb
- Integration of a MERN application for the first time
- Project scope
- Data source, currently the project uses its own data
- Importing bigger sets of data for testing

## Future Implementions
- Unit tests
- Edge cases, for example, when 1 or 2 parameters are used or no product match
- Typescript
- Integration with e-commerce, ability to buy the product

## Installation 

1. Clone repo
2. Run yarn install on project folder
3. Make sure that you follor [this](https://www.mongodb.com/developer/how-to/use-atlas-on-heroku/) tutorial to set up Mongo 
4. Create a .env file and type the connection with your Mongo Atlas database
5. Import data, see example products.csv
